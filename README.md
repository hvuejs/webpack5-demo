## webpack 5 安装

> 前言：在之前我们都是手动配置搭建webpack项目，webpack官方提供了cli快速构建基本模版，无需像之前一样手动配置entry、plugins、loader等

1. 新建文件夹 (webpack5-demo)

2. ```npm --init -y``` 生成 ```package.json``` 文件

3. 安装 ```webpack``` 和 ```webpack-cli```
```base
npm install webpack webpack-cli -D
```

4. 安装第3步后，命令行执行
```base
npx webpack init
```

5. 根据提示选择配置即可
